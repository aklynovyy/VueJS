1.Обработанное сообщение
Создайте две переменные и создайте обработанное свойство, которое будет писать заданную строку в обратном порядке.

2.Создание предложения
Создайте Vue.js объект, который будет объединять в одно слово две переменные в объекте. Это необходимо сделать при помощи обработанных свойств.






























































































1.Для начала давайте будет выводить все в HTML документе:
<div id="app">
  <p>Изначальное сообщение: "{{ message }}"</p>
  <p>Обработанное сообщение: "{{ reversedMessage }}"</p>
</div>
Теперь добавить простой код Vue.js, который будет ссылаться на первоначальную переменную и при помощи встроенных функций писать её в обратном порядке:
new Vue({
  el: '#app',
  data: {
    message: 'Она не жена, но...'
  },
  computed: {
    reversedMessage: function () {
      return this.message.split('').reverse().join('')
    }
  }
});



2.В результате у вас должен был получиться следующий код:
new Vue({
  el: '#app',
  data: {
    firstName: 'Nikola',
    lastName: 'Tesla'
  },
  computed: {
    fullName: function () {
      return this.firstName + ' ' + this.lastName
    }
  }
});
